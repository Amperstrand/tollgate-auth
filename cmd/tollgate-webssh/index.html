Ecash for SSH access — try it in your browser

How it works

  1. Get a free Cashu token from the testnut faucet (fake Bitcoin, zero value).
  2. Click Connect to open an SSH session in your browser.
  3. You get 1 minute per satoshi. An 8-sat token = 8 minutes of shell time.
  4. Choose Timer mode (countdown) or Adventure mode (Crypt of Cashu dungeon crawler).

New to Cashu? It's a Bitcoin ecash system. Tokens are bearer instruments — anyone with the token string can redeem them. Learn more. You can also get tokens from the faucet or any Cashu wallet.
