What is this?

A pay-per-minute SSH server that accepts Cashu ecash tokens instead of passwords. You paste a token as your SSH username. The server redeems it, creates a throwaway guest account, and gives you an interactive bash shell for as many minutes as the token is worth (1 sat = 1 minute). When time runs out or you disconnect, the account is destroyed.

Tokens are from testnut — a fake mint for testing Cashu wallets. They are not real Bitcoin and have zero monetary value. Guests get real shell access on a real server. Read the security disclaimer.
